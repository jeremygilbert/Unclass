‘Responsive Styling I’

Topic: CSS

Date: May 13, 2013

Time: 1-1:50 p.m.

Location: Fisk 311

Lead instructor: Jeremy Gilbert

Description: Responsive design detects which devices users are connecting with, allowing designs to be optimized across different devices. Learn adaptive techniques and break points.

Learning Goals: (at least three, no more than five)

  1. Know what it means for an HTML-layout to be responsive
  2. Know why to make it responsive
  3. Learn how to make HTML flexible
  4. Learn how to write a media query

Pre-work: (readings, tutorials and/or assignments)

  2. A Book Apart, Mobile First, ‘Layout’
