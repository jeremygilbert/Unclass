‘CSS Style I’

Topic: ‘CSS’

Date: April 14, 2013

Time: 2-2:50PM

Location: Fisk 311

Lead instructor: Jeremy Gilbert

Description: The basics of CSS styling--how to access and affect HTML elements with internal and external CSS styling. We'll cover basic selectors, properties, and values to style elements on a webpage.

Learning Goals: (at least three, no more than five)

  1. What can you style?
  2. How are styles written?
  3. What is the cascade?
  4. What are some basic type styles?
